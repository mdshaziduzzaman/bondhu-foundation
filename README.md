# 🏢 Foundation Management System

A full-stack web application for managing a foundation's financial activities including member deposits, expenses, and real-time balance tracking.

🔗 Live Demo: https://bbb.free.nf/app/login?i=1

---

## 🚀 Features

### 👥 Member Management
- Secure login system (User ID & Password)
- Role-based access control
- Individual member dashboard

### 💰 Financial Tracking
- Monthly member deposit tracking
- Expense management system
- Real-time total balance calculation

### 📊 Dashboard & Reports
- Overview of total funds
- Deposit history per member
- Expense history
- Clean and user-friendly UI
